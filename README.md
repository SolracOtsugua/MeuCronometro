Cronômetro Digital em C# ⏱️

Este repositório contém uma aplicação desktop funcional de um cronômetro com precisão de milissegundos. 

Este projeto foi desenvolvido inteiramente com fins educacionais, servindo como um exercício prático de treinamento durante o meu curso no SENAI.

🎯 Objetivo do Projeto

O objetivo principal foi praticar o desenvolvimento de interfaces gráficas e a manipulação de eventos e tempo utilizando a classe Stopwatch e componentes visuais do Windows Forms.

✨ Funcionalidades

Iniciar e Pausar: Controle do tempo real com precisão de milissegundos.

Zerar: Reiniciar o contador interno e limpar a tela.

Registrar Voltas: Função para salvar o tempo exato da volta em uma lista visível na interface (ListBox), ativada apenas enquanto o cronômetro está rodando.

🛠️ Tecnologias e Ferramentas Utilizadas

Linguagem: C#;

Framework: Windows Forms (.NET);

IDE: Visual Studio;

Controle de Versão: Git / GitHub;

🚀 Como Executar o Projeto
Se você quiser baixar e testar este projeto localmente na sua máquina, siga os passos abaixo:

Certifique-se de ter o Visual Studio instalado com a carga de trabalho "Desenvolvimento para desktop com .NET".

Clone este repositório para a sua máquina.

Navegue até a pasta do projeto e dê um duplo clique no arquivo da solução (MeuCronometro.sln).

Com o Visual Studio aberto, pressione a tecla F5 no seu teclado ou clique no botão Iniciar na barra superior.
